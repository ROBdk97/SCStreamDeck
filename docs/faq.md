# FAQ

## Where is the download?

Use the `Download Simple` or `Download Advanced` button on [Home](index.md), or go to [Download & Installation](install.md) for direct links and package details.

## Do I need a GitHub account?

No. You can download the plugin without an account.

## How do I update?

Go to [Update & Uninstall](update.md). Current release builds require you to uninstall before installing a newer version.

## Which package should I install?

Use **Simple** if you just want the plugin to work without installing anything else.  
Use **Advanced** if you want the smaller plugin package and are happy to install the **.NET 10 Desktop Runtime x64** first.

## Which languages does the plugin support?

The plugin UI currently supports English, German, French, and Spanish.  
By default it follows the detected language, and you can override it from the `Control Panel` action.

## Which Star Citizen channels are supported?

`LIVE`, `HOTFIX`, `PTU`, `EPTU`, and `TECH-PREVIEW`.

## I want to create my own theme. How?

See [Themes](themes.md).

## I found a bug. Where do I report it?

Use GitHub Issues if possible:

- [https://github.com/ROBdk97/SCStreamDeck/issues](https://github.com/ROBdk97/SCStreamDeck/issues)

Include:

- Plugin version
- Stream Deck version
- What you expected vs what happened
- Screenshots (if UI-related)
